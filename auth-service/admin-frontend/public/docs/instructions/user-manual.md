# User Manual

Welcome to the Auth Service Admin Panel. This guide will help you understand how to use all features of the application.

## Overview

The Auth Service Admin Panel is a centralized management system for:
- **User Management** - Create, edit, activate/deactivate users
- **Role Management** - Configure roles and their permissions
- **Entity Management** - Manage applications and their configurations
- **Session Management** - Monitor and revoke active sessions
- **Mailing Management** - View email queue and status

## Getting Started

### Login

1. Navigate to the login page
2. Enter your username and password
3. Click "Sign In"

> **Note:** Only users with ADMIN role can access this panel.

### Navigation

Use the sidebar on the left to navigate between different sections:
- **Dashboard** - Overview statistics
- **Users** - User management
- **Roles** - Role configuration
- **Entities** - Application management
- **Sessions** - Active session monitoring
- **Mailings** - Email queue
- **Settings** - Personal preferences

## Dashboard

The dashboard provides a quick overview of:
- Total users count
- Active sessions
- Recent activity
- System health status
