# User Management

This guide explains how to manage users in the Auth Service Admin Panel.

## Viewing Users

1. Click on **Users** in the sidebar
2. The user list displays all registered users
3. Use filters to search by username, email, or status

## Creating a New User

1. Click the **Add User** button
2. Fill in the required fields:
   - Username (unique)
   - Email
   - First Name
   - Last Name
   - Password
3. Select user status
4. Click **Create**

## Editing Users

1. Find the user in the list
2. Click the **Edit** icon
3. Modify the desired fields
4. Click **Save**

## Managing User Roles

1. Find the user in the list
2. Click **Add Role** icon
3. Select the role and application code
4. Click **Add**

To remove a role:
1. Click the **X** on the role chip
2. Confirm the removal

## Activating/Deactivating Users

1. Find the user in the list
2. Click the status toggle or edit the user
3. Change the status field
4. Save changes

## Resetting Passwords

1. Find the user in the list
2. Click the **Reset Password** icon
3. Enter the new password
4. Click **Reset**

## Best Practices

- Always use strong passwords
- Assign minimum necessary roles
- Regularly review user accounts
- Deactivate users who no longer need access
