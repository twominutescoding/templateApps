# Version 1.0.0

**Release Date:** 2026-02-04

## New Features

### Core Application
- Full-stack business application template
- Spring Boot 4.0.1 backend
- React 19.2 frontend
- JWT-based authentication

### User Interface
- Modern Material-UI v7 design
- Responsive layout
- Dark/Light theme toggle
- Custom color palettes

### Data Tables
- Server-side pagination
- Advanced filtering
- Column sorting
- Inline editing
- Bulk edit mode
- Export to CSV/Excel

### Settings
- Theme customization
- Date format preferences
- Timestamp format preferences
- Profile management

### Demo Features
- Demo Products CRUD example
- Component showcase
- Chart examples

## Technical Details

- **Frontend**: React 19.2, TypeScript, Vite
- **UI Library**: Material-UI v7
- **State Management**: Jotai
- **Backend**: Spring Boot 4.0.1, Java 17
- **Database**: H2 (dev), Oracle (prod)
- **Authentication**: JWT tokens via external auth-service
