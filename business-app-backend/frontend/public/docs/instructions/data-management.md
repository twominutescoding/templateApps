# Data Management

This guide explains how to work with data tables in the Business Application.

## Table Overview

Our data tables provide powerful features for managing your data:
- Server-side pagination
- Column sorting
- Advanced filtering
- Inline editing
- Bulk operations
- Data export

## Viewing Data

### Sorting

Click on any column header to sort:
- First click: Ascending order
- Second click: Descending order
- Third click: Remove sorting

### Filtering

Use the filter row above the table:
- **Text fields**: Type to search (partial match)
- **Number fields**: Enter exact values
- **Date fields**: Use date pickers for ranges
- **Select fields**: Choose from dropdown options

### Pagination

Control how data is displayed:
- Change rows per page (10, 25, 50, 100)
- Navigate using Previous/Next buttons
- Jump to specific pages

## Editing Data

### Inline Editing

1. Click the **Edit** icon on any row
2. Modify the fields directly in the table
3. Click **Save** to confirm or **Cancel** to discard

### Bulk Editing

1. Click **Bulk Edit** button
2. Select rows to edit using checkboxes
3. Modify values in selected rows
4. Click **Save All** to save all changes

## Exporting Data

### Export to CSV

1. Click the **Export** button
2. Select **CSV** format
3. File downloads automatically

### Export to Excel

1. Click the **Export** button
2. Select **Excel** format
3. File downloads automatically

## Best Practices

- **Filter first**: Apply filters before editing to work with smaller datasets
- **Save frequently**: Save changes regularly when doing bulk edits
- **Check before delete**: Always verify before deleting records
- **Use exports**: Export data as backup before major changes
